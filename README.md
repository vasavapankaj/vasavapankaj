<details>
<summary><b>Hazard Water Detector — IoT Water Quality Monitoring System</b></summary>

<br/>

An ESP32-based water quality monitoring prototype designed to analyze TDS and turbidity parameters and provide real-time safety status.

| Engineering Metric | Details |
|:---|:---|
| **Stack** | ESP32, Embedded C++, PHP, MySQL, JavaScript |
| **Scale** | Multi-sensor real-time monitoring prototype |
| **Performance** | Continuous sensor acquisition and status evaluation |
| **Security** | Structured server-side data storage |
| **Impact** | Supports rapid identification of unsafe water conditions |
| **Repository** | [GitHub Profile](https://github.com/vasavapankaj) |

### Engineering Overview

The system integrates **TDS and turbidity sensors with an ESP32 microcontroller** to monitor water quality parameters.

Sensor readings are processed against configured safety thresholds. The system displays real-time status and uses LED and buzzer indicators when unsafe conditions are detected.

A PHP and MySQL-based monitoring dashboard provides structured storage and visualization of water readings.

**Tech:** `ESP32` `IoT` `C++` `TDS Sensor` `Turbidity Sensor` `PHP` `MySQL`

</details>
